# Image Assets Required

Please download the following images from https://pestdial.net and place them in this directory:

## Required Images:

1. **logo.png** - Company logo (header)
2. **hero-banner.jpg** - Main hero section background image
3. **cockroach-control.png** - Service icon for cockroach control
4. **termite-control.png** - Service icon for termite control  
5. **rodent-control.png** - Service icon for rodent control
6. **bed-bug-control.png** - Service icon for bed bug control
7. **ant-control.png** - Service icon for ant control
8. **mosquito-control.png** - Service icon for mosquito control
9. **about-team.jpg** - Team photo for about section
10. **phone-icon.png** - Phone icon for contact section
11. **email-icon.png** - Email icon for contact section
12. **location-icon.png** - Location icon for contact section
13. **facebook-icon.png** - Facebook social media icon
14. **instagram-icon.png** - Instagram social media icon
15. **twitter-icon.png** - Twitter social media icon
16. **youtube-icon.png** - YouTube social media icon

## Image Specifications:

- **Logo**: 200x60px (PNG with transparent background)
- **Service Icons**: 80x80px (PNG with transparent background)
- **Hero Banner**: 1200x600px (JPG, high quality)
- **About Team**: 600x400px (JPG)
- **Contact Icons**: 24x24px (PNG with transparent background)
- **Social Icons**: 32x32px (PNG with transparent background)

## How to Download:

1. Visit https://pestdial.net
2. Right-click on each image
3. Select "Save image as..."
4. Save to this directory with the exact filenames listed above
5. Ensure images are optimized for web (compressed but high quality)

## Alternative:

You can also use placeholder images temporarily by:
1. Using placeholder services like placeholder.com
2. Creating simple colored rectangles as placeholders
3. Using free stock photos from unsplash.com or pexels.com

The website will work with placeholder images, but for the best result, use the actual images from the original site.
